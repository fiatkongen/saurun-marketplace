# saurun-marketplace

Personal Claude Code plugin marketplace by Rasmus Hove Munk.

## Installation

```bash
# Add the marketplace
/plugin marketplace add fiatkongen/saurun-marketplace

# Install the plugin
/plugin install saurun@saurun-marketplace
```

## Included Plugin

### saurun

Personal Claude Code skills, commands, and agents including:

- **Skills**: codex-bridge, plan-fixer, long-run-implement
- **Commands**: /long-run
- **Agents**: product-manager, system-architect, ux-ui-designer, and more
