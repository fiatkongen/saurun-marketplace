# Test Deploy Skill

This is a simple test skill used for validating the deploy-to-marketplace workflow.

## Purpose

- Copies skills correctly
- Validates structure
- Handles version bumping

## When to Use

This skill is only for testing deployment workflows. It has no other purpose.
